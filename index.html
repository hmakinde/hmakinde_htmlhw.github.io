    Summary: Latitude Analysis Dashboard

    In building this dashboard, I created individual pages for each plot and a means by which we can navigate between them. These pages contain the visualizations and their corresponding explanations.

    We have a landing page, a page where we can see a comparison of all of the plots, and another page where we can view the data used to build them. I used the CSV table as well as the images provided.

    This is an analysis and visualization of weather of 700+ cities across the world. The main objective was to showcase the relationship of various factors like temperature, humidity, cloudiness and wind speed with respect to the distance from the equator. Variables analyzed include:

    • Temperature (F) vs. Latitude
    • Humidity (%) vs. Latitude
    • Cloudiness (%) vs. Latitude
    • Wind Speed (mph) vs. Latitude
